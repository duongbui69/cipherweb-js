# CipherWeb JS

## Backend

```bash
cd backend
npm install
npm run dev
```

API: http://localhost:8080/api/crypto

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Deploy

- Backend: Render/Railway, start command: `npm start`
- Frontend: Vercel, env: `VITE_API_URL=https://your-backend-url`
